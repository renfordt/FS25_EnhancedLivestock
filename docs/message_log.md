# Message Log

The Message Log keeps track of important events in your livestock operation.

## Overview

Enhanced Livestock records significant events that affect your animals, allowing you to track what's happening on your farm.

## Event Types

### Birth Events
- New animals born
- Number of offspring
- Parent information
- Genetics of newborns

### Death Events
- Animal deaths
- Cause of death (old age, disease, complications)
- Animal details

### Disease Events
- New infections
- Disease spread
- Recovery notifications
- Treatment status

### Transaction Events
- Animals bought
- Animals sold
- Purchase prices
- Sale prices

### Breeding Events
- Successful breeding
- Pregnancy confirmations
- Due date notifications

### Herdsman Events
- Automated purchases
- Automated sales
- Castrations performed
- Names assigned

## Accessing the Log

Access the message log through:

1. The main farm menu
2. The husbandry management screen
3. Keyboard shortcut (configurable)

## Log Features

### Filtering
Filter messages by:

- Event type
- Animal type
- Date range
- Husbandry location

### Sorting
Sort messages by:

- Date (newest/oldest first)
- Event type
- Animal type

### Details
Click on any message to see:

- Full event details
- Animal information
- Related events

## Notification Settings

Configure which events trigger notifications:

| Event Type | Default |
|------------|---------|
| Births | On |
| Deaths | On |
| Diseases | On |
| Sales | Off |
| Purchases | Off |
| Herdsman Actions | Off |

## Tips

!!! tip "Using the Message Log"
    - **Check daily** - Review the log regularly to catch issues early
    - **Track diseases** - Monitor disease spread patterns
    - **Verify Herdsman** - Confirm the AI is making good decisions
    - **Record keeping** - Use the log for long-term herd management planning
