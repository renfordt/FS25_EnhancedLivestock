## Reproduction

### Fertility Cycle

| Age | Fertility Status |
|-----|-----------------|
| 0-12 months | Not fertile |
| 12-24 months | Coming into fertility |
| 24-60 months | Peak fertility |
| 60-84 months | Declining |
| 84-132 months | Low fertility |
| 132+ months | Not fertile |

### Pregnancy
- **Gestation period:** 10 months
- **Litter size:** 1-3 calves (average 1)
- **Minimum health for breeding:** 75%

## Production

### Milk
Only lactating cows produce milk.

| Age | Milk Production (liters/day) |
|-----|------------------------------|
| 0-12 months | 0 |
| 12-36 months | 180 |
| 36-72 months | 255 (peak) |
| 72-120 months | Declining to 110 |
